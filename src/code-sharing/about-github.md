## Github Gist

GitHub Gist 有点类似一个云剪贴板，可以用来托管代码片或者 Markdown 等标记语言书写的小型文档。

Gist 的方便之处在于分享十分方便，可以随时修改（浏览器就行），还能记录迭代过程，每个 Gist 就是一个 Git 管理的小仓库。

Gist 分为公开的和私有的，私有的 Gist 不会在用户的主页显示，也无法被搜索引擎搜索到，但指向 Gist 的链接所有用户都可以访问的。
